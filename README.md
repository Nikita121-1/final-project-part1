README: WeatherInsight — приложение для анализа погодных данных
Описание приложения
WeatherInsight — это веб-приложение для визуализации и анализа метеорологических данных. Приложение позволяет исследовать температурные тренды, осадки и другие погодные показатели по различным городам.

Основные функции
Загрузка и обработка данных из локальной базы данных SQLite

Фильтрация данных по городу и временному периоду

Визуализация данных через интерактивные графики

Прогноз погоды на заданный период

Сравнительный анализ показателей между городами

Разведочный анализ данных (EDA)

Использование приложения
Боковая панель (настройки)
Выбор города из выпадающего списка доступных городов

Настройка периода наблюдений через выбор дат

Параметры отображения таблицы (количество строк на странице)

Настройки графиков для EDA и сравнительного анализа

Прогноз погоды на 7-30 дней

Основные разделы
Общая статистика — показывает количество записей и городов в базе

Данные за период — таблица с фильтрацией по городу и датам

Прогноз погоды — простой прогноз температуры на заданный период

EDA — исследование распределения выбранных показателей

Сравнение городов — визуализация различий между городами
